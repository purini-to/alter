# Alter
勉強用に作成
「MEAN stack」を使用してチャットアプリを作成する

# 環境構築
## Vagrant使用の場合(おすすめ)
Vagrantの使用できる環境で以下のコマンドを入力する
```
git clone https://github.com/purini-to/alter
cd alter
vagrant up
```
Provisionが動作して、「MEAN stack」の環境が整う

# 実行方法
GitHubからソースをクローンする
```
git clone https://github.com/purini-to/alter
```
プロジェクトフォルダに移動後、gulpタスクを実行する
```
cd alter
gulp
```
